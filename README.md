# 📊 Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow. It includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard creation, and report generation. The objective is to transform raw data into meaningful insights that support data-driven decision-making.

---

## Dataset

The dataset was imported into Python for preprocessing and analysis. After cleaning and validating the data, it was used for SQL queries and Power BI visualizations.

---

## Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis
* **Pandas & NumPy** – Data manipulation and preprocessing
* **Matplotlib & Seaborn** – Data visualization
* **PostgreSQL / MySQL / SQL Server** – SQL querying and analysis
* **Power BI** – Interactive dashboard development
* **Microsoft Word / PDF** – Project report
* **Gamma** – Presentation (PPT) creation

---

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data structure and data types.

### 2. Exploratory Data Analysis (EDA)

* Analyzed dataset dimensions and summary statistics.
* Identified missing values and outliers.
* Explored relationships between variables using visualizations.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into a SQL database.
* Wrote SQL queries to extract meaningful insights.
* Used filtering, grouping, joins, aggregate functions, and sorting for analysis.

### 5. Power BI Dashboard

* Connected the processed data to Power BI.
* Built interactive dashboards with charts, KPIs, slicers, and filters.
* Presented key metrics in a user-friendly format.

### 6. Report & Presentation

* Documented the analysis, findings, and recommendations.
* Created a presentation using Gamma to summarize the project.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook or Python script.
4. Run the data loading, EDA, and data cleaning steps.
5. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
6. Execute the SQL queries.
7. Open the Power BI file (`.pbix`) and refresh the data.
8. Review the report and Gamma presentation.
